¿Qué son las alertas básicas?
Las alertas básicas de Bootstrap se utilizan para mostrar mensajes con diferentes colores que indican el tipo de información: éxito, advertencia, información, error, entre otros. El uso de clases como .alert-primary, .alert-secondary, etc., determina el color y estilo de cada alerta.

Alertas con Botón de Cierre

¡Acción completada correctamente!
Hubo un error en el proceso.
Advertencia: Cambios sin guardar.
¿Qué son las alertas con botón de cierre?
Las alertas con botón de cierre permiten al usuario cerrar la alerta una vez que ha leído el mensaje. Esto se consigue con la clase .alert-dismissible y un botón con la clase .btn-close. El atributo data-bs-dismiss="alert" permite ocultar la alerta al hacer clic en el botón.

Alertas con Iconos

¡Acción exitosa!
Error crítico. Por favor revisa el sistema.
¿Qué son las alertas con iconos?
Se pueden añadir iconos a las alertas para representar visualmente el tipo de mensaje. Por ejemplo, un icono de éxito (bi-check-circle) para un mensaje positivo y un icono de error (bi-exclamation-circle) para un mensaje de alerta. Los iconos de Bootstrap están disponibles como SVG y se añaden con la clase d-flex y align-items-center.

¿Qué son las alertas con enlaces?
Las alertas pueden incluir enlaces para proporcionar más detalles o para redirigir al usuario a otras páginas. Para añadir un enlace dentro de la alerta, se utiliza el componente .alert-link que estiliza el enlace de forma adecuada dentro de la alerta.

¿Qué son las alertas personalizadas?
Las alertas personalizadas permiten aplicar animaciones y estilos especiales. En este caso, hemos utilizado una animación CSS llamada fadeIn que hace que la alerta aparezca suavemente. Las animaciones pueden mejorar la interacción visual de los usuarios con las alertas.
