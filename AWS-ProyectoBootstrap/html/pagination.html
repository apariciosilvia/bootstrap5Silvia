¿Qué es la Paginación en Bootstrap 5?
La paginación en Bootstrap 5 es un componente que permite dividir contenido en varias páginas de forma visualmente atractiva y fácil de navegar.
  • Paginación Simple: Muestra enlaces básicos para navegar entre páginas.
  • Tamaños: Personaliza el tamaño de los elementos de la paginación con clases como .pagination-lg y .pagination-sm.
  • Estados Activos y Deshabilitados: Usa las clases .active y .disabled para resaltar la página actual o deshabilitar enlaces.


OBJETIVO DEL ELEMENTO:

El componente Pagination en Bootstrap 5 se usa para crear paginaciones estilizadas en listas o tablas de datos, permitiendo a los usuarios navegar entre múltiples páginas de contenido. Facilita la navegación en aplicaciones con gran cantidad de datos, mejorando la usabilidad y accesibilidad.
