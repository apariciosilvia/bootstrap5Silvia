Ejemplo 2: Imagen con Sombra

Aquí usamos la clase .shadow-lg para agregar una sombra pronunciada, dándole un efecto flotante a la imagen.

Ejemplo 3: Imagen Redonda Completa

Ejemplo 4: Imagen Miniatura

Este ejemplo utiliza la clase .img-thumbnail, que agrega un borde y sombra ligera para dar un estilo de miniatura.

Ejemplo 5: Imagen Fluida

La clase .img-fluid asegura que la imagen sea completamente responsiva y se ajuste al ancho del contenedor.

Ejemplo 6: Imagen Rectangular con Bordes Rectos

En este ejemplo se utiliza la clase .w-100 para que la imagen ocupe el 100% del ancho del contenedor, manteniendo sus bordes rectos.


OBJETIVO DEL ELEMENTO:

El objetivo de Images en Bootstrap 5 es proporcionar clases predefinidas para imágenes de forma responsiva y estilizada. Permiten ajustar tamaños, bordes, alineaciones y hacer que las imágenes se adapten automáticamente a diferentes dispositivos, mejorando la presentación visual.
