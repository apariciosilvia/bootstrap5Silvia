¿Qué son los Button Groups?
Los Button Groups son una forma de agrupar botones relacionados visualmente. Puedes usar la clase .btn-group para agrupar botones de forma horizontal, o la clase .btn-group-vertical para alinearlos verticalmente. También puedes agregar funcionalidades adicionales, como menús desplegables, para incluir más opciones dentro de un grupo de botones. Son útiles cuando tienes varias acciones posibles para un mismo contexto.

Button Group Básico

¿Qué es un Button Group Básico?
Un Button Group Básico agrupa varios botones juntos para hacerlos más compactos y visualmente relacionados. Este tipo de agrupación ayuda a mantener los botones organizados y mejora la interfaz de usuario, especialmente cuando tienes varias acciones similares que deben realizarse en una sola línea. Se crea utilizando la clase .btn-group.

Button Group Vertical

¿Qué es un Button Group Vertical?
Un Button Group Vertical es una variante del Button Group, pero en lugar de organizar los botones horizontalmente, los organiza en una columna. Es útil cuando tienes muchas acciones y quieres que los botones estén alineados verticalmente, lo que es ideal para dispositivos móviles o para interfaces donde el espacio horizontal es limitado. Se utiliza con la clase .btn-group-vertical.

¿Qué es un Button Group con Dropdown?
Este tipo de Button Group incluye un botón que despliega un menú de opciones adicionales cuando el usuario hace clic en él. Es útil cuando tienes muchas opciones para ofrecer, pero no deseas que todos los botones se muestren en la interfaz. Para crear este tipo de botón, usamos un botón con la clase .dropdown-toggle y un menú desplegable con la clase .dropdown-menu.

Button Group con Iconos

¿Qué es un Button Group con Iconos?
Los Button Groups con iconos combinan la funcionalidad de los grupos de botones con la visibilidad de los iconos. Esto proporciona una experiencia más visual, lo que ayuda a los usuarios a identificar rápidamente la acción asociada con cada botón. En el ejemplo se utilizaron iconos como bi-heart-fill, bi-pencil-fill, y bi-check-circle.

OBJETIVO DEL ELEMENTO:

El objetivo de Button Groups en Bootstrap 5 es agrupar múltiples botones relacionados en una sola línea o bloque, manteniendo un diseño uniforme y compacto. Se usan en barras de herramientas, selecciones de opciones o controles de acciones, mejorando la organización y usabilidad.
