Descripción de los Badges en Bootstrap 5

Los Badges en Bootstrap son pequeñas etiquetas que pueden mostrarse sobre botones, en enlaces o en cualquier lugar para mostrar un número, un estado, una notificación o simplemente una etiqueta informativa.

Estos marcadores son perfectos para representar estados, como el número de notificaciones, mensajes, o cualquier otro tipo de información numérica o textual que quieras destacar.

Para crear un Badge, puedes usar la clase .badge y aplicarla sobre el elemento donde deseas mostrarlo.

OBJETIVO DEL ELEMENTO:

El objetivo de Badges en Bootstrap 5 es mostrar etiquetas pequeñas y destacadas para resaltar información clave, como notificaciones, estados o contadores. Se usan comúnmente en botones, encabezados o listas para mejorar la visualización de datos importantes.
