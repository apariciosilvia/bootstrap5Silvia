Descripción de Componentes de la Tabla

  • Clases de Estilo: Bootstrap ofrece varias clases para estilizar tablas:
    • .table: Clase base para aplicar estilos básicos a la tabla.
    • .table-striped: Aplica un patrón de rayas alternas en las filas para mejorar la legibilidad.
    • .table-hover: Destaca las filas al pasar el cursor sobre ellas.
    • .table-bordered: Agrega bordes a cada celda.
    • .table-dark y .table-light: Ofrecen esquemas de color oscuro o claro para las tablas.
  • Encabezados: Utilizando la clase .table-dark en el thead, se aplica un diseño oscuro al encabezado.
  • Cuerpo de la Tabla: Cada fila (tr) representa un registro, y puedes usar clases personalizadas como .table-success o .table-danger para resaltar información importante.
  • Pie de Tabla: El tfoot se utiliza para mostrar un resumen o información adicional, como el promedio general.

OBJETIVO DEL ELEMENTO:

Las tablas en Bootstrap 5 permiten mostrar datos de forma estilizada y responsiva con clases predefinidas. Ofrecen diseños como **striped, bordered, hover** y permiten personalizar colores, alineación y tamaños sin necesidad de CSS adicional. Son ideales para reportes y dashboards.
