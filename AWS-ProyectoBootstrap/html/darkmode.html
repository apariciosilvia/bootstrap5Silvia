Modo Oscuro con Bootstrap 5

Este ejemplo muestra cómo alternar entre modos claro y oscuro utilizando data-bs-theme.

¿Cómo Usar Modo Oscuro en Bootstrap 5?

  1. Incluye Bootstrap 5 en tu proyecto.
  2. Utiliza el atributo data-bs-theme en el contenedor principal para definir el tema inicial.
  3. Usa JavaScript para alternar entre light y dark.
  4. Personaliza los estilos según el tema usando variables de CSS de Bootstrap.

El modo oscuro es útil para mejorar la experiencia visual del usuario en entornos con poca luz.



OBJETIVO DEL ELEMENTO:

En Bootstrap 5 no hay un modo oscuro nativo, pero se pueden aplicar estilos oscuros con clases como .bg-dark, .text-light y componentes como .table-dark. También se puede implementar Dark Mode manualmente con CSS y JavaScript, permitiendo a los usuarios alternar entre modos claro y oscuro.
