Modales Básicos

Título del Modal
Este es un modal básico de Bootstrap 5.
¿Qué es un modal básico?
Un modal básico de Bootstrap es una ventana emergente que se utiliza para mostrar contenido adicional sin abandonar la página actual. Puedes personalizar el título, el contenido y los botones dentro del modal. Se activa mediante el atributo data-bs-toggle="modal".

Modales con Variaciones

Modal Grande
Este es un modal con un tamaño más grande, ideal para contenido más amplio o detallado.
¿Qué son los modales con variaciones de tamaño?
Puedes cambiar el tamaño de los modales utilizando las clases modal-lg para modales grandes o modal-sm para modales pequeños. Esto te permite ajustar el diseño según el contenido que necesites mostrar.

Un ejemplo de su uso podria ser: Confirmación antes de eliminar Cuando un usuario intenta eliminar un elemento, se puede abrir un modal para confirmar la acción.


OBJETIVO DEL ELEMENTO:

El objetivo de Modal en Bootstrap 5 es mostrar ventanas emergentes interactivas para contenido adicional sin salir de la página. Se usan para formularios, confirmaciones, alertas o detalles, mejorando la experiencia sin recargar la interfaz.
